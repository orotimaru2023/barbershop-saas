# ⚡ Quick Start: Configurar Keycloak (30 minutos)\n\n## 🎯 Objetivo\nConfigurar OAuth2 realm e roles para autenticação.\n\n---\n\n## OPÇÃO A: Keycloak Cloud Managed (Recomendado)\n\n### Passo 1: Criar Conta (5 min)\n\n1. Abra: https://www.keycloak.cloud\n2. Clique **\"Sign Up\"**\n3. Preencha:\n   - Email\n   - Password (forte)\n   - Name\n4. Confirme email\n5. ✅ Login\n\n### Passo 2: Criar Realm (5 min)\n\n1. No dashboard, clique **\"Create Organization\"**\n2. Nome: `barbershop`\n3. Clique **\"Create\"**\n4. ✅ Realm criado\n\n### Passo 3: Criar OAuth2 Client (5 min)\n\n1. Na sidebar, vá para **\"Clients\"**\n2. Clique **\"Create Client\"**\n3. Nome: `barbershop-web`\n4. Selecione: **\"Web application\"**\n5. Clique **\"Create\"**\n6. Na próxima tela:\n   - **Redirect URIs:** `http://localhost:3000/callback`\n   - **Web Origins:** `http://localhost:3000`\n7. Clique **\"Save\"**\n8. ✅ Client criado\n\n### Passo 4: Copiar Credenciais (5 min)\n\n1. No client `barbershop-web`, vá para **\"Credentials\"**\n2. Copie:\n   - **Client ID** → será `NEXT_PUBLIC_KEYCLOAK_CLIENT_ID`\n   - **Client Secret** → será `KEYCLOAK_CLIENT_SECRET`\n3. Note também:\n   - **Realm Name:** será `NEXT_PUBLIC_KEYCLOAK_REALM` = `barbershop`\n   - **URL:** será `NEXT_PUBLIC_KEYCLOAK_URL` = `https://xxx.keycloak.cloud`\n\n### Passo 5: Criar Roles (5 min)\n\n1. Na sidebar, vá para **\"Roles\"**\n2. Clique **\"Create Role\"** (4 vezes):\n   - `owner` - Dono da barbearia\n   - `admin` - Administrador\n   - `barber` - Barbeiro\n   - `client` - Cliente\n3. Para cada role:\n   - Nome\n   - Descrição (opcional)\n   - Clique **\"Create\"**\n4. ✅ 4 roles criadas\n\n---\n\n## OPÇÃO B: Keycloak Self-Hosted (Local)\n\n### Pré-requisito: Docker instalado\n\n```bash\n# 1. Baixar e rodar Keycloak\ndocker run -p 8080:8080 \\\n  -e KEYCLOAK_ADMIN=admin \\\n  -e KEYCLOAK_ADMIN_PASSWORD=admin \\\n  quay.io/keycloak/keycloak:latest \\\n  start-dev\n\n# Vai levar 1-2 minutos para iniciar\n```\n\n### Acessar\n1. Abra: http://localhost:8080\n2. Login:\n   - User: `admin`\n   - Password: `admin`\n3. ✅ Admin console aberto\n\n### Criar Realm\n\n1. Canto superior direito, clique **\"Master\"** dropdown\n2. Clique **\"Create Realm\"**\n3. Nome: `barbershop`\n4. Clique **\"Create\"**\n5. ✅ Realm criado\n\n### Criar Client\n\n1. Na sidebar esquerda, clique **\"Clients\"**\n2. Clique **\"Create\"**\n3. Client ID: `barbershop-web`\n4. Clique **\"Create\"**\n5. Na aba **\"Settings\"**:\n   - **Valid Redirect URIs:** `http://localhost:3000/*`\n   - **Web Origins:** `http://localhost:3000`\n6. Clique **\"Save\"**\n7. ✅ Client criado\n\n### Copiar Credenciais\n\n1. Vá para aba **\"Credentials\"**\n2. Copie:\n   - **Client ID**\n   - **Client Secret**\n\n### Criar Roles\n\n1. Na sidebar, clique **\"Roles\"**\n2. Clique **\"Create Role\"** (4 vezes):\n   - `owner`\n   - `admin`\n   - `barber`\n   - `client`\n3. Para cada um:\n   - Nome\n   - Clique **\"Save\"**\n4. ✅ 4 roles criadas\n\n---\n\n## Passo 6 (Ambas Opções): Atualizar .env.local\n\nAbrindo `.env.local` novamente:\n\n```env\n# Supabase (já preenchido)\nNEXT_PUBLIC_SUPABASE_URL=https://xxx.supabase.co\nNEXT_PUBLIC_SUPABASE_ANON_KEY=eyJ...\nSUPABASE_SERVICE_ROLE_KEY=eyJ...\n\n# Keycloak (PREENCHER AGORA)\nNEXT_PUBLIC_KEYCLOAK_URL=https://xxx.keycloak.cloud\n# OU para local: http://localhost:8080\n\nNEXT_PUBLIC_KEYCLOAK_REALM=barbershop\nNEXT_PUBLIC_KEYCLOAK_CLIENT_ID=barbershop-web\nKEYCLOAK_CLIENT_SECRET=xxx\n```\n\nSalve!\n\n---\n\n## ✅ CHECKLIST: Keycloak http://localhost:3000/api/health\n```\n\n---\n\n## 🆘 Troubleshooting\n\n### Problema: \"Cannot find Client\"\n- Verificar se Client ID está correto em .env.local\n- Verificar se Client Secret está correto\n\n### Problema: \"Realm not found\"\n- Verificar se realm name é `barbershop`\n- Verificar se está usando a URL certa\n\n### Problema: \"CORS error\"\n- Se Keycloak local: testar em http://localhost:8080\n- Se Keycloak cloud: verificar Web Origins no client\n\n---\n\n## 📊 Comparação: Cloud vs Local\n\n| Aspecto | Cloud | Local |\n|--------|-------|-------|\n| Setup | 15 min | 10
